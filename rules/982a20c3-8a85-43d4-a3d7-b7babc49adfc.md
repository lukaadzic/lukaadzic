To perform specific tasks, use the following commands: bun run --filter=@pennyone/database codegen (codegen), bun run --filter=@pennyone/backend typecheck (typecheck of backend code), and bun run eval:proactive:quality (eval of proactive quality). Additionally, for web-specific checks, use bun run --filter=@pennyone/web typecheck.
